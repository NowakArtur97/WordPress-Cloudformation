## General info

AWS CloudFormation template with required resources and commands to install and run Wordpress on an EC2 instance.

## Built With

Cloudformation resources:

- VPC
- Internet Gateway
- VPC Gateway Attachment
- Route Table
- Route
- Subnet Route Table Association
- Subnet
- Security Group
- EC2 Instance
- DB Subnet Group
- DB Instance
